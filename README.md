# GLO Digital Media Art Gallery

This is a custom-built React gallery for **GLO**. It is designed to look bold and high-end on desktop monitors while staying easy to use on mobile phones.

## How to Use
- **Filters:** Use the "All", "Images", and "Videos" buttons to sort the art.
- **View Details:** Click any piece of art to see it and a detailed description.
- **Inquire:** Use the form inside the pop-out to send an email inquiry via Formspree.
- **Call GLO:** Click the phone icon in the top right to call directly.

## Main Components
- **index.css**: A simple setup file that ensures the app fills the screen
- **App.jsx**: Controls the main gallery layout, the desktop/mobile scaling, and the filter logic.
- **About.jsx**: Contains the Artist Statement and the story behind the work.
- **data.json**: This is where all the art titles, descriptions, and image links are stored.

## Deployment
This site is designed to be hosted on **Vercel**. Any changes pushed to your GitHub repository will automatically update the live website.
